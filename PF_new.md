# Yash Satish Patil - Portfolio

This is the source code for my personal portfolio website, built with HTML, CSS, and JavaScript.

## Project Structure

- **`index.html`**: The main structure of the website.
- **`style.css`**: All the custom styling for the portfolio.
- **`README.md`**: Project description and setup instructions.
- **`Images/`**: (You need to create this folder) Contains all academic records, certificates, and the profile picture.
    - `yp profile.jpg` (Profile Picture)
    - `10th yp.jpg` (10th Marksheet)
    - `12th yp.jpg` (12th Marksheet)
    - `3rd yp.jpg` (Sem 3 Grade Sheet)
    - `4th yp.jpg` (Sem 4 Grade Sheet)
    - `college yp.jpg` (ProQuest Certificate)
    - `delloit yp.jpg` (Deloitte Certificate)
    - `ibm yp.jpg` (IBM Certificate)
    - `Gym yp.png` (Project Image 1)
    - `Gym1 yp.png` (Project Image 2)

## Technologies Used

- **HTML5** (Structure)
- **CSS3** (Styling and Responsiveness)
- **JavaScript** (Section navigation)

## How to View

Simply open the `index.html` file in any modern web browser.

## Deployment

This project is intended to be deployed using **GitHub Pages** for easy, public accessibility.
